### Installation

```
npm install && bower install
```

### Tests

```
gulp test
```

### Server Start

Build
```
gulp build
```

Lokal über gulp 
```
gulp serve
```

oder für Serve aus dem dist
```
gulp serve:dist
````

Alternativ
```
npm start
```
